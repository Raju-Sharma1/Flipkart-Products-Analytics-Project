# Flipkart Product Analytics – SQL Data Warehouse and Analytics Project

## Overview
This project demonstrates an end-to-end **data analytics workflow** using Python and SQL Server to analyze Flipkart product listing data. The objective is to transform raw product data into a structured analytical dataset and extract business insights related to **sales performance, pricing strategy, customer ratings, and seller performance**.

The project simulates a real-world analytics pipeline including **data preprocessing, data warehouse setup, exploratory analysis, business intelligence queries, and advanced SQL analytics**.

## Dataset Description

The dataset contains Flipkart product listings with information about pricing, discounts, seller details, ratings, and sales performance.

| Column | Description |

# 1. Data Preprocessing (Python)

Raw data was cleaned and validated using **Pandas** before loading into the data warehouse.

### Key Steps

#### Dataset Exploration
- Checked dataset shape
- Inspected column data types
- Generated statistical summaries

#### Handling Missing Values
- Missing values in the `size` column were filled using the **mode value**.

#### Duplicate Detection
- Checked for duplicate rows in the dataset.

#### Data Formatting
- Rounded `shipping_weight_g` to **2 decimal precision**.
- Standardized `payment_modes` values by replacing commas with `|`.

Example:
  COD,CARD → COD|CARD


#### Data Quality Validation
- Verified **minimum and maximum values** for all numeric columns.

#### Output
Cleaned dataset exported as:
  "**Cleaned_Flipkart_Products_Data.csv**"

# 2. Data Warehouse Setup (SQL Server)

A dedicated analytics database was created. "**Flipkart_Analytics**"


### Table Created
"**dbo.Flipkart_Products**"


The table stores cleaned product data with optimized SQL data types such as:

- `DECIMAL` for pricing values
- `INT` for counts
- `DATE` for listing dates
- `NVARCHAR` for categorical attributes

### Data Loading

Data was imported using **BULK INSERT** for efficient loading of the CSV dataset into SQL Server.

# 5. Advanced SQL Analytics

### Product Ranking by Category

