# Seller Dashboard (Operational Analytics)

## Overview
The **Seller Dashboard** focuses on analyzing seller performance, operational efficiency, and inventory availability using the Flipkart product dataset. This dashboard provides insights into how sellers perform in terms of ratings, delivery efficiency, sales contribution, and stock distribution across product categories.

The goal of this dashboard is to evaluate seller-level operational metrics and identify patterns in seller performance across cities and product categories.

---

## Key Metrics

The dashboard highlights three high-level KPIs that summarize overall seller performance:

- **Average Seller Rating:** 4.00  
- **Average Delivery Time:** 6.01 days  
- **Total Revenue:** 4.76T  

These KPIs provide a quick overview of seller quality, delivery efficiency, and overall revenue generated within the dataset.

- Kolkata

# Dashboard Objective

The **Seller Dashboard** helps answer important analytical questions such as:

- Which sellers generate the **highest revenue**?
- Which sellers handle the **largest order volumes**?
- How efficient are sellers in terms of **delivery performance**?

This dashboard complements the **Sales Dashboard** and **Product Dashboard**, providing a broader perspective on **marketplace performance and seller contribution**.
