# Seller Dashboard

The **Seller Dashboard** focuses on analyzing seller performance, delivery efficiency, and marketplace contribution within the dataset.  
It helps evaluate how different sellers perform based on revenue generation, order volume, and delivery timelines.

---

# Seller Performance KPIs

The top section of the dashboard provides key metrics that summarize seller activity across the platform.

### Total Sellers
Displays the **total number of sellers** present in the dataset.

### Total Orders
Represents the **total number of orders fulfilled by sellers**.

### Total Revenue
Shows the **overall revenue generated by all sellers combined**.

### Average Delivery Time
Indicates the **average delivery time required to complete orders**.

These KPIs provide a quick overview of seller activity and operational performance across the marketplace.

---

# Average Delivery Time by Seller

**Visual Type:** Column Chart

This visualization shows the **average delivery time for each seller**.

### Purpose
- Compare delivery efficiency across sellers.
- Identify sellers with faster or slower delivery performance.

### Example Insights
- Sellers with lower delivery times demonstrate **better logistics efficiency**.
- Higher delivery times may indicate **potential operational bottlenecks**.

---

# Revenue by Seller

**Visual Type:** Bar Chart

This chart displays the **total revenue generated by each seller**.

### Purpose
- Identify top-performing sellers based on revenue.
- Compare seller contributions to overall marketplace sales.

### Example Insights
- Some sellers contribute a **significant portion of total revenue**.
- Helps identify sellers that drive the **largest share of marketplace transactions**.

---

# Seller Performance Table

**Visual Type:** Table

This table provides a **detailed comparison of sellers** using key performance metrics.

### Fields Displayed
- Seller Name
- Total Revenue
- Total Orders

### Purpose
- Provide a structured view of seller performance.
- Allow quick comparison between sellers based on revenue and order volume.

### Example Insights
- Sellers with both **high revenue and high order volume** typically indicate strong marketplace presence.
- Enables identification of **top-performing sellers**.

---

# Dashboard Objective

The **Seller Dashboard** helps answer important analytical questions such as:

- Which sellers generate the **highest revenue**?
- Which sellers handle the **largest order volumes**?
- How efficient are sellers in terms of **delivery performance**?

This dashboard complements the **Sales Dashboard** and **Product Dashboard**, providing a broader perspective on **marketplace performance and seller contribution**.
